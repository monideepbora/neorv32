Detailed Description

"Intrinsic" library for the NEORV32 bit manipulation (B.Zbb) extension. Also provides emulation functions for all intrinsics (functionality re-built in pure software).

Author
Stephan Nolting
Warning
This library is just a temporary fall-back until the B/Zbb extensions are supported by the upstream RISC-V GCC port.
